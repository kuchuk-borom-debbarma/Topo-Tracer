# Minimal Blur Frontend Theme

Restyle the frontend toward a quiet glass/blur theme with fewer colors and less explanatory copy.

## Steps

- Add compact blur theme overrides in frontend CSS.
- Remove or hide secondary labels/descriptions where they add noise.
- Keep core navigation, trace search, pagination, graph controls, and auth actions usable.
- Run frontend build and smoke-check served page.
