  <files>sdks/node-js/src/BatchExporter.ts</files>
  <action>
    Implement `BatchExporter` class (implements `ITelemetryExporter`):
    - Buffer events in an array.
    - Use `setInterval` to flush at `batchIntervalMs` (default 5 min).
    - `export(event)`: pushes event to buffer. If `buffer.length >= maxQueueSize`, drop oldest or stop accepting (configurable).
    - `flush()`: 
      - POSTs current buffer to `collectorUrl` using `fetch`.
      - Clears buffer.
      - Handles network errors gracefully (optional: basic retry logic).
    - `shutdown()`: stops the interval and calls `flush()`.
  </action>
