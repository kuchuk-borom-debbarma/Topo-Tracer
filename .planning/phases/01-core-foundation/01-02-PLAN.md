<task type="auto">
  <name>Task 2: Implement Span Class</name>
  <files>sdks/node-js/src/Span.ts</files>
  <action>
    Implement `Span` class with:
    - Properties: `id`, `traceId`, `name`, `startTime`.
    - Constructor: 
      - Generates `id` via `crypto.randomUUID()`.
      - Inherits `traceId` from parent or generates new one.
      - Emits `IngestNodeStart` event via `Tracer.exportEvent`.
      - If parent exists, emits `IngestEdgeStart` from parent to this span.
    - `end(message?)`: 
      - Sets end time.
      - Emits `IngestNodeEnd` event.
    - `recordData(data)`: appends to internal metadata.
    - `startChild(name, options)`: helper to start a nested span.
  </action>
  <verify>
    <automated>npx tsc sdks/node-js/src/Span.ts --noEmit --esModuleInterop --module esnext</automated>
  </verify>
