    Implement `Tracer` class with:
    - Private `AsyncLocalStorage<Span>` instance.
    - `init(config: TracerConfig)` to setup collector and exporter.
    - `startSpan(name, options, fn)`: 
      - Detects parent from ALS if not provided in options.
      - Creates new `Span`.

    - `addEdge(fromSpan, toSpan, label?)`: (per FR-3) Emits an `IngestEdgeStart` event from `fromSpan` to `toSpan`.
    - `exportEvent(event)`: proxies to the configured exporter.
  </action>

<task type="auto">
  <name>Task 2: Implement Span Class</name>
  <files>sdks/node-js/src/Span.ts</files>
  <action>
    Implement `Span` class with:
    - Properties: `id`, `traceId`, `name`, `startTime`, `importanceLevel`.
    - Constructor: 
      - Generates `id` via `crypto.randomUUID()`.
      - Inherits `traceId` from parent or generates new one.
      - Emits `IngestNodeStart` event via `Tracer.exportEvent`.
      - If parent exists, automatically emits `IngestEdgeStart` from parent to this span (implicit edge).
    - `end(message?)`: 
      - Sets end time.
      - Emits `IngestNodeEnd` event.
    - `recordData(data)`: appends to internal metadata.
    - `startChild(name, options)`: helper to start a nested span (uses `tracer.startSpan`).
  </action>
  <verify>

  </verify>
