## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Caller-to-repository | Caller supplies `userId`, `traceId`, and threshold; repository enforces caps internally. |
| Repository-to-ClickHouse | Query params become scoped latest-node SQL. |
| Projection-to-materialization | Bounded projection method must not reuse full latest read model loading. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-NODES | Denial of Service | visible-node projection query | mitigate | Enforce `DEFAULT_PROJECTION_NODE_CAP + 1`, slice to cap, return `capHit`. |
| T-04-SCOPE | Elevation of Privilege | node query scope | mitigate | Assert `user_id` and `trace_id` filters and query params. |
| T-04-FULL | Denial of Service | accidental full read | mitigate | Source assertion forbids `loadLatestReadModel` inside bounded node method. |
</threat_model>
