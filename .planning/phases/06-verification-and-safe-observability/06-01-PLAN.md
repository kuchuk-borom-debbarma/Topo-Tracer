<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Event bus to worker | Worker receives unknown event payload batches and chooses trace rebuild work. |
| Worker to materializer | Worker delegates trace-level rebuilds without owning persistence. |
| Materializer to repository | Materializer trusts repository to return ordered rows after the authoritative checkpoint. |
| Test fake to production contract | Stateful fakes must model the contract without inventing production broker semantics. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-01-DUP | Tampering | duplicate delivery | mitigate | Assert replay does not mutate latest read state/checkpoint beyond the first successful materialization. |
| T-06-01-ORDER | Repudiation | event ordering contract | mitigate | Document and test that ordering is bus-owned; do not add hidden materializer sorting. |
| T-06-01-LATE | Integrity | checkpoint boundary | mitigate | Assert no post-checkpoint raw rows results in no writes. |
| T-06-01-DIAG | Availability | malformed timing | mitigate | Assert diagnose-and-continue behavior instead of trace-wide failure. |
</threat_model>
