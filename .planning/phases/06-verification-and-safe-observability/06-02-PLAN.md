## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Materializer to logs | Materializer emits operational summaries from raw/read model work. |
| Projection service to logs | Projection service emits read summary metadata from bounded graph results. |
| Telemetry payloads to observability | Node/edge/event data can contain user-controlled or sensitive fields. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-02-LEAK | Information Disclosure | materializer logs | mitigate | Replace full summary/object logging with scalar fields and test forbidden keys. |
| T-06-02-PROJ | Information Disclosure | projection logs | mitigate | Extend projection log assertions to forbid raw graph/event keys. |
| T-06-02-OBS | Repudiation | runtime diagnosis | mitigate | Include safe ids, counts, durations, caps, and diagnostic counts. |
| T-06-02-REG | Tampering | future log edits | mitigate | Source assertions fail if raw log keys are reintroduced. |
</threat_model>
