<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Bounded input-to-projector | Projector trusts only capped latest read-model input, not raw events or database rows. |
| Original graph-to-projected graph | Explicit edges are transformed into visible/ghost projected edges. |
| Malformed data-to-output | Orphan or malformed edges are omitted and counted rather than invented. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-HIDDEN | Tampering | visibility logic | mitigate | Tests prove hidden nodes are never returned as normal nodes. |
| T-05-EDGE-DOS | Denial of Service | snapped edges | mitigate | Aggregate duplicate snapped edges by source, target, and type. |
| T-05-MALFORMED | Tampering | orphan edges | mitigate | Omit malformed edges and increment metadata instead of inventing endpoints. |
| T-05-DATALEAK | Information Disclosure | projector metadata | mitigate | Return counts/ranges only for ghosts, not hidden raw payload arrays. |
</threat_model>
